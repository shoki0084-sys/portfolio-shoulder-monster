Meet Minutes AI

Google Meet 議事録自動生成

会議の録音・チャットから、文字起こし→構造化された議事録→Google Docs保存までを自動化するPythonツールです。

画面キャプチャ

誰の課題

オンライン会議後に議事録を手作業でまとめる時間がかかり、決定事項やアクションアイテムの共有が遅れがちになるチーム・個人事業主。

何をしたか

Google Meetの録音ファイル（MP4等）をGemini APIで文字起こしし、チャットログ（テキスト）と合わせて議事録形式（日時・参加者・議題・決定事項・アクション）に要約。オプションでGoogleドキュメントに保存し、共有リンクを取得できるCLIツールを開発しました。

主な機能

  • 録音ファイルの文字起こし（MP4 / WebM / MP3 等）
  • チャットログ併用による議事録要約
  • ターミナル表示または Google Docs へ自動保存
  • OAuth 2.0 による Google Docs API 連携

使用技術

PythonGemini APIGoogle Docs APIOAuth 2.0

学び

「録音を取る」だけで終わらせず、社内共有までつなぐ一連のフローを設計することで、ジムのスタッフ会議やクライアントMTGにも応用できる土台になりました。
