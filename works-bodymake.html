AIボディメイク管理システム

クライアントの「続ける」を支える、記録・分析・通知の一元化システムです。

画面キャプチャ

誰の課題

パーソナルジム・オンラインコーチで、食事・体重・トレーニングの記録がLINEやメモに分散し、指導前にデータを集める手間がかかっている。

何をしたか

Webブラウザから食事・体重・筋トレを入力すると、Dify（AI）が分析・アドバイスを返します。データはGoogleスプレッドシートに自動保存され、体重はグラフで可視化。必要に応じてLINEへアドバイスをプッシュし、定時の自動通知にも対応しています。

主な機能

  • 食事記録の入力とAI分析結果の保存
  • 体重・体脂肪率の記録とChart.jsによるグラフ表示
  • 筋トレ種目・回数・セット数の記録
  • Difyによるアドバイス生成（手動表示・LINE送信）
  • 日本時間での定時LINE通知（設定でON/OFF可能）
  • Googleスプレッドシートへのシート別自動追記

使用技術

Node.jsExpressDify API LINE Messaging APIGoogle Sheets APIChart.js

設計・運用のポイント

Dify無料枠を意識し、定時通知の無効化や食事入力のまとめ方など、運用コストに配慮した設計をREADMEに明記しています。事業者がスプレッドシートを直接確認できるため、データの透明性も確保しています。

学び・今後の拡張

「記録する」だけでなく「返事が返ってくる」「LINEで届く」まで一連で設計することで、クライアントの継続率向上に直結する構成を学びました。複数トレーナー・複数クライアント対応、管理画面の追加などにも展開可能です。
