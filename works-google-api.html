制作実績 / Google API

Google Workspace Automation

Google API 連携ツール群

記録アプリやBotと組み合わせる土台として、Google周りのAPI連携経験をまとめています。

実装一覧

  • Googleスプレッドシート — 食事・体重・筋トレデータの追記・読取（本番利用）
  • Googleカレンダー — イベントの登録・管理
  • Gmail API — メール送信
  • Googleドライブ — ファイルのアップロード
  • Googleドキュメント — ドキュメントの新規作成
  • Google Meet — ミーティング作成
  • Google Maps — ルート・距離・時間（ツーリングプランナーで利用）

使用技術

PythonNode.jsOAuth 2.0サービスアカウント

学び

「全部つなぐ」より、事業者がすでに使っているGoogleツールから接続する方が、導入ハードルが低いと実感しました。
